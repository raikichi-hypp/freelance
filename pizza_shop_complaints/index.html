1. ピザを楽しみにしていた夜

金曜日の夜、仕事が終わった私は「今日はピザを頼もう！」と決めていた。

スマホでピザ屋のアプリを開き、お気に入りの「ダブルチーズマルゲリータ」と「ガーリックバタークラスト」を注文。 「30分でお届け」と書かれている。ワクワクしながら映画をセットして待っていた。

2. 予想外のトラブル発生！

しかし、約束の30分を過ぎてもピザは届かない…。 40分…50分…ついに1時間が経過。

「これはおかしい！」と、私はピザ屋に電話した。

  • 📞 電話対応はそっけない：「配達が遅れてます」
  • 🕒 新しい到着時間の説明なし：「もうすぐ着きます」
  • 😡 謝罪がない：「混んでるので仕方ないです」

ついにピザが届いたが、ピザは冷めていて、具も片寄っていた。 私の楽しい夜は台無し…。このまま泣き寝入りするのか？

3. クレーム対応のプロとは？

そこで、私はもう一度電話をかけることにした。 すると、今度は別のスタッフが出て、驚くほどプロフェッショナルな対応をしてくれた。

✅ 1. すぐに謝罪する

📢 「ご迷惑をおかけして申し訳ありません！」

最初のスタッフとは違い、誠実な謝罪をしてくれた。 これだけで、私の怒りは少し和らいだ。

✅ 2. 問題を正しく把握する

👂 「状況を詳しく教えていただけますか？」

私が「遅れた上に冷めていた」と伝えると、しっかりと聞いてくれた。

✅ 3. 解決策を提示する

🍕 「すぐに新しいピザをお届けするか、割引クーポンをお渡しできます」

選択肢を用意してくれたことで、私は「次回また頼もう」と思えた。

✅ 4. 感謝の気持ちを伝える

💖 「貴重なご意見をありがとうございます。今後の改善に活かします！」

この一言があるだけで、私は「このお店は信頼できる」と感じた。

4. まとめ：信頼を築く接客術

今回の経験から、クレーム対応で大切なのは次の4つのポイントだと分かった。

  • ✅ 最初に誠実な謝罪をする
  • ✅ お客様の話をしっかり聞く
  • ✅ 解決策を提示して、選択肢を与える
  • ✅ 最後に感謝の気持ちを伝える

もしピザ屋の副業を始めるなら、「クレーム対応の質」が お店の評判を大きく左右することを覚えておこう。
